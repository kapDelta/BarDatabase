BarDatabase

База данных предназначена для автоматизации учёта в баре. Она охватывает следующие бизнес-процессы: 
- Учёт сотрудников и занимаемых должностей 
- Расчёт заработной платы (смешанная система: оклад + процент от продаж) 
- Учёт меню (позиции товаров и напитков) 
- Фиксация заказов клиентов и расчёт итоговой выручки

Структура базы данных 
База данных состоит из 5 основных таблиц, связанных между собой внешними ключами. 

Таблица positions (Должности) 
Хранит перечень должностей, существующих в баре. 

Таблица employees (Сотрудники) 
Содержит личные данные и информацию о занятости сотрудников 

Таблица menu (Меню) 
Перечень товаров (напитков и блюд), которые продаются в баре 

Таблица orders (Заказы) 
Фиксирует факт продажи (закрытые чеки) 

Таблица order-items (Состав заказа) 
Детализация каждого заказа (что именно и в каком количестве купили) 

Расчёт заработной платы 
Заработная плата сотрудника за период (например, месяц) рассчитывается по формуле: 
Итоговая ЗП = Оклад + (Личный вклад в выручку * Процент сотрудника / 100) 
Где: 
- Оклад (base salary) – берётся из таблицы positions. 
- Процент сотрудника (bonus percentage) – берется из таблицы positions. 
- Личный вклад в выручку – это сумма всех total amount из таблицы orders, где employee id равен ID сотрудника, а order date попадает в расчётный период.
